# web-dev-template

#### A template app which houses sub-apps developed during the web-dev course

#### Install Instructions

1. git clone https://github.com/tintin1343/web-dev-template.git
2. cd web-dev-template
3. npm install
4. mongod
5. node server.js
6. browse to localhost:3000
